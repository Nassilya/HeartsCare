# 🌸 HeartsCare - Assistant Intelligent pour la Santé Hormonale

**Hearts & Care** est une application web moderne et bienveillante dédiée à la santé hormonale et au bien-être féminin. Elle intègre un chatbot intelligent propulsé par un modèle de langage de pointe (LLM) exécuté localement pour garantir la confidentialité totale des données de santé des utilisatrices.

---

## 🛠️ Technologies Utilisées
* **Frontend/Backend :** Python, Streamlit
* **Moteur IA :** Ollama, Llama 3 (8B)
* **Gestion de version :** Git & GitHub

---
